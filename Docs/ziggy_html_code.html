CBIT App · Request/Response Architecture
── INPUT ──
Step 1 · Input
User Speaks
Child or caregiver speaks into device microphone
Step 2 · Transcription
Apple On-Device Speech Recognition
Audio → text, fully on-device, never leaves the phone
COPPA safe no network call zero latency
── INBOUND GUARDS ──
🚨 CRISIS EXIT
Suicide / self-harm keywords → immediate crisis resources, conversation ends
Step 3 · Inbound Guard A
PII Scrubber
Strips / masks before anything is sent to any external API
full names addresses phone / email school names doctor names insurance IDs
⚠️ LOG + STRIP
PII detected → masked in transcript, flagged in admin log, processing continues
🛑 OUT OF SCOPE
Redirect response generated — no Claude call, no RAG, no cost
Step 4 · Inbound Guard B
Out-of-Scope Classifier
Catches medical, legal, diagnostic, and irrelevant queries before RAG
medication names dosage / mg diagnose me is this a tic side effects diet / weight unrelated topics
✅ IN SCOPE
Passes to RAG pipeline — normal flow continues
── RETRIEVAL ──
Step 5 · Embedding
Voyage AI — Embed Query
Scrubbed + sanitized text → 1024-dim vector (voyage-3 model)
inputType: query voyage-3
Step 6 · Retrieval
Supabase pgvector — Filtered Semantic Search
Cosine similarity search filtered by session stage + age group + tic type metadata. Returns top 3–5 chunks.
session_stage filter age_group filter tic_type filter top-k: 5
── AI REASONING ──
Step 7 · Language Model
Claude API — claude-sonnet-4-6
System prompt assembled with: Ziggy persona + age voice profile · Retrieved doc chunks (source grounding) · Session context (stage, tic, CR, history) · Conversation history · Sanitized user message
RAG grounded first protocol rails age-tuned voice streaming
── OUTBOUND GUARDS ──
🔄 REGENERATE
Blocked content detected → regenerate with stronger constraint or safe fallback
Step 8 · Outbound Guard
Response Safety Filter
Scans Claude's response before it ever reaches the user
medication names leaked diagnostic language PII echoed back therapy claims harmful content tone vs age check
✅ CLEAN
Response passes all checks → proceeds to voice synthesis
── OUTPUT ──
Step 9 · Voice Synthesis
ElevenLabs — Streaming TTS
Clean text → audio stream using age-appropriate custom Ziggy voice profile
young child voice older child voice teen voice adult voice eleven_turbo_v2
Step 10 · Delivery
Audio Plays to User
Streamed audio plays in real time as tokens generate. Conversation history updated. Session state machine advanced if protocol milestone detected.
history updated state machine tick reward trigger check
Guard Layer Summary
🔴 Inbound PII Scrubber
Runs on-device before any external API call. Masks real names, addresses, school/doctor names, IDs. Logs detection events.
🟢 Out-of-Scope Classifier
Keyword + intent check. Catches meds, diagnosis requests, crisis language. Returns scripted safe response — no API cost.
🔴 Outbound Safety Filter
Scans Claude response before delivery. Catches leaked meds, diagnostic claims, PII echoed back, therapy impersonation.
🟣 RAG Grounding
Retrieved CBIT docs injected before Claude reasons. Clinical facts come from your vetted library — not model hallucination.
